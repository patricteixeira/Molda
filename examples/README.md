# Exemplos públicos

Os exemplos deste diretório usam licença MIT e servem como fixtures portáteis
para implementações externas. `brand-package-reference/` é a saída mínima de um
adapter M4: tokens DTCG, logo nativo e manifesto de integridade.

Valide a fixture sem executar código do adapter:

```bash
brandrt package-validate examples/brand-package-reference
```

O exemplo é deliberadamente incompleto como marca: não redistribui fonte nem
manual. Ao entrar no wizard, essas lacunas continuam explícitas e exigem
confirmação humana.
