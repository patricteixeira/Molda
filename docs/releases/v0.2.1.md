# A mesma criação, agora portátil entre Windows e Linux

Molda v0.2.1 é uma correção de distribuição para a linha v0.2. Ela preserva
integralmente o catálogo autoral, as recomendações de marca e o Modo Carrossel
da v0.2.0, mas garante que o entrypoint shell do container web continue
executável quando o repositório é obtido em Windows.

### O que este corte corrige

- força finais de linha LF em todos os arquivos `*.sh`, independentemente da
  configuração global `core.autocrlf` do operador;
- impede o ciclo de reinício do container web causado pelo shebang convertido
  para `/bin/sh\r`;
- amplia o gate estático da release para recusar uma regra Git ausente ou
  qualquer script shell versionado com CRLF;
- comprova a distribuição por checkout Windows, build Docker, healthchecks e
  resposta HTTP da aplicação.

Não há mudança de contrato de produto, de Brand IR, de composição, de conteúdo
ou de formato de exportação em relação à v0.2.0. Esta versão substitui a
v0.2.0 como corte recomendado da linha v0.2.

### Limites conhecidos

- A distribuição continua self-hosted, single-tenant e sem contas individuais;
  exposição pública exige TLS e uma camada externa de acesso.
- O importador Figma, a instância pública multi-tenant e o add-in PowerPoint
  continuam fora deste corte.
- O round-trip PPTX permanece comprovado em Google Slides, LibreOffice e testes
  estruturais; PowerPoint Desktop continua como gate aberto.
- Fontes comerciais não são distribuídas. Preview e fidelidade final dependem
  das licenças e instalações disponíveis ao operador.
- OCR e tradução rodam localmente, mas materiais escaneados de baixa qualidade
  ainda podem exigir confirmação humana durante a instalação da marca.
